# **App Name**: BashConnect

## Core Features:

- Party Creation: Users can create new party events, providing essential details like name, date, time, and description.
- Guest Invitation Link: Generate a unique, shareable link for each party that guests can use to join and participate without authentication.
- Real-time Party Chat: Enable a 2-way, real-time chat interface for all invited party participants within the event.
- Party Details View: View all relevant party information and a list of participants who have joined the event.
- AI Activity Suggestion Tool: Leverage an AI tool to suggest engaging party activities based on party type, guest age, and preferences provided by the host.

## Style Guidelines:

- Primary color: A vibrant yet welcoming purple (#8C3CE1) to represent connection and celebration.
- Background color: A very subtle, light lilac (#F5F1FA) to ensure readability and a clean aesthetic in the light theme.
- Accent color: A lively medium blue (#5E5EE1) to highlight interactive elements and provide contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly technical and bold feel. Body font: 'Inter' (sans-serif) for excellent readability in chat and longer descriptions.
- Utilize simple, clean outline icons related to parties, chat, calendar, and user profiles to maintain a consistent modern visual language.
- Employ a responsive and intuitive layout with clear divisions for party details, participant lists, and the real-time chat interface to ensure ease of use on any device.
- Incorporate subtle animations for UI feedback, such as message sending, button presses, and transitions between party views, to enhance the interactive experience without being distracting.
